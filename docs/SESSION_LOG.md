# SESSION_LOG.md

This file tracks all work sessions for this project.

Agents must read the last 2 entries at the start of every session.
Agents must append a new entry at the end of every session.

---

## Entry template

```markdown
---

## [YYYY-MM-DD] — [AGENT NAME]

**Session scope:** what was requested

**What was done:**
- item 1
- item 2

**Decisions made:**
- why X was done this way (critical context for the next agent)

**Files modified:**
- path/to/file.ts
- path/to/other.ts

**Session status:** what is pending / what is the next step
```
